Security FAQ
Is my API key safe? expand_more
Yes. Keys are encrypted using AES-256 before transmission. We do not store keys permanently on our relay servers; they are only used for session handshake validation.
Why am I rate limited? expand_more
Riot's API has strict rate limits. If you've made too many requests recently (error 429), the system pauses automatically to prevent an IP ban. Please wait for the cooldown timer.
